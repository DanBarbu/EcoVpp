Turn a condominium into a Virtual Power Plant.

White-label, full-stack orchestration platform that lets multi-unit residential buildings share rooftop solar under RED II Collective Self-Consumption, dispatch flexibility on local markets, and settle every kWh on-chain — running on a sub-€20 ESP32 + HaLow node.

See live demo → View source on GitHub

What it does

Six microservices, one Helm chart, end-to-end from a sub-€20 hardware node to on-chain settlement.

⚡ Sub-€20 ingest

ESP32-S3 + ADE7913 + 802.11ah HaLow. Telemetry POSTed with HaLow-tolerant retry; OCR confidence < 0.80 routes to manual verification.

☀️ RED II sharing

15-minute pro-rata Collective Self-Consumption allocator. Surplus auto-listed as offers on the GSY-e P2P market.

🎚️ Flexibility dispatch

FlexMeasures price/congestion → SET_LOAD_LIMIT over MQTT to EV chargers and heat pumps. Cloud-to-edge < 3 s.

🔗 On-chain settlement

Every batch hashed and anchored on Energy Web Origin. NFT Guarantees of Origin minted per asset+batch. PII stays off-chain.

📈 Forecasting

Hourly Open-Meteo GTI → DC kW forecast pushed via FlexMeasures v0.31 trigger/poll API.

🛡️ Resilient

Helm chart with HPA, K6 load test (1k req/s), Chaos Mesh experiments. RTO target < 30 s.
